How to use KWIC search:

About KWIC search:

KWIC search is a Key Word In Context search program, in which you identify the website, file, or wikipedia article you would like to search, and the keyword (search term) you would like to look for.
Please indicate if you are looking at a website, file, or wikipedia article.
In the left-hand box you will see the all the sentences in which the keyword occurs, and the keyword will be displayed in bold.
Once you select a sentence, the table on the right-hand side will display the form, lemma, and POS tag of each word in the sentence.

URL/Filename:

This field should contain the URL/Filename/Wikipedia term you would like to search for.
URL: The URL you want to search, including http:// or https://
e.g. https://docs.oracle.com/javase/tutorial/uiswing/components/tooltip.html
File: The file you want to search, including the whole path to find it on your computer.
e.g. C:\Users\User1\Corpus\AECorpus.txt
Wiki: Enter the name of a wikipedia article you would like to search for; if multiple options are possible, a box will pop up with possible articles to choose from. Keep in mind: searching on wikipedia is case-senstive.
e.g. Berlin
If "German" is selected as the language, German Wikipedia will be searched.

Search term:

This field should contain the word you would like to search for.
(e.g. see)
Keep in mind: The search is case-sensitive.

POS:

Select a POS tag if you would like to search for a keyword with a specific POS tag.
If the POS tag box is left blank, the program will search for all instances of the keyword. e.g. if you search for "see" with POS tag VB, it will only return instances in which "see" is the base form of the verb.
If you search for "see" without a POS tag, it will return all instances of "see".
To see what the different POS tags mean, look at the "POS tag meanings" tab.

N-grams:

The number of context words (words on either side of the keyword) you would like to be displayed.
The default n-gram is the whole sentence, so the whole sentence will be displayed.

Search button:

To search for the keyword (optionally with a specific POS tag) in the given URL/File, press the search button.

Language options:

KWIC search works in both English and German, but is in English as a default.
When German is selected, German OpenNLP tools are used, so the search term, POS tags, lemmas, etc. will be displayed as their German equivalents.

Statistics button:

Once a search has been completed, you can click the "statistics" button to find out:
how many instances of the keyword occur in the text
how many sentences the keyword is in
the distribution of the word over the POS tags it has
the time it took to complete the search

Save button:

Pressing this button prompts the user for a filename, in which to save the results
of the search.
If you attempt to save before you conduct a search, the program will show an error message.

Clear all button:

Pressing this button will clear all search boxes, results, tags, and reset the POS and N-gram boxes.

Help button:

Pressing this button will show the help pages for the program, including how to use the program, and the meanings of the POS tags.
